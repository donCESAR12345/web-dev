What's this webpage?

This webpage is a homework assigned in a university course called "Development of web applications".

The objective of the homework is to give an introduction to the markup language HTML, going straight to the newest version HTML5, basically we (the students) were told to use as many HTML tags as possible while maintaining a coherent and cohesive design.

Design?

Yes, design, we were told to make a beautiful page without using CSS, unfortunately (or not?) for me, I started making the page the same day as our first class, prior to receiving the Google Classroom assignment with the instructions and requirements.

So that same day I saw my raw HTML only webpage looking kinda gross, feeling mildly disgusted by this I searched across the whole internet (first DuckDuckGo result probably) on how to stylize a webpage using CSS, and fortunately for me, it was pretty easy, even if time consuming I really wanted my page to look at least acceptable, but with some of these tricks to make the webpage useful based on JavaScript interactions, I decided to just do it with CSS and let the JS topic out of scope for now.

But why a personal page?

I thought about it for some time (at least 12.9 seconds) and decided to go with the personal page because I've seen some of these linked in GitHub repositories or other code forges, these looked nuts, I knew I wasn't going to have such a beautiful outcome, but thinking long term, this will surely be useful for me, as I'll probably have one of these webpages in the future.
